# Code Block

Syntax-highlighted code display with copy support.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/code-block.json
```

## Import

```tsx
import { CodeBlock } from '@vllnt/ui'
```

## Usage

```tsx
<CodeBlock>CodeBlock content</CodeBlock>
```
