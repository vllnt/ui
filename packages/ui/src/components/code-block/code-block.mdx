# Code Block

Syntax-highlighted code display with copy support.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/code-block.json
```

## Import

```tsx
import { CodeBlock } from '@vllnt/ui'
```

## Usage

```tsx
<CodeBlock>CodeBlock content</CodeBlock>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `children` | `ReactNode` | No | Code content. Strings, numbers, arrays, and nested element text are extracted for rendering and copy. |
| `code` | `string` | No | Explicit code string. Preferred over `children` when provided. |
| `language` | `string` | No | - |
| `showLanguage` | `boolean` | No | `true` or `false` |
