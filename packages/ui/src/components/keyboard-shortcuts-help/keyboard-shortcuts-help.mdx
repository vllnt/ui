# Keyboard Shortcuts Help

Displays available keyboard shortcuts to the user.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/keyboard-shortcuts-help.json
```

## Import

```tsx
import { KeyboardShortcutsHelp } from '@vllnt/ui'
```
