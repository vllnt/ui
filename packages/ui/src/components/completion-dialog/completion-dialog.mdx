# Completion Dialog

Dialog displayed upon completing a task or workflow.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/completion-dialog.json
```

## Import

```tsx
import { CompletionDialog } from '@vllnt/ui'
```
