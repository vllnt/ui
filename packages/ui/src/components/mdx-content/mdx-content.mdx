# MDX Content

Renders MDX content with component mapping.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/mdx-content.json
```

## Import

```tsx
import { MdxContent } from '@vllnt/ui'
```

## Usage

```tsx
<MdxContent
  content="value"
>
  MdxContent content
</MdxContent>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `components` | `Record&lt;string, React.ComponentType&lt;any&gt;&gt;` | No | - |
| `content` | `string` | Yes | - |
| `enableMDX` | `boolean` | No | `true` or `false` |
