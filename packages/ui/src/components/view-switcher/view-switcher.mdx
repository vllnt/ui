# View Switcher

URL param-based toggle between named views with pill/tab styling.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/view-switcher.json
```

## Import

```tsx
import { ViewSwitcher } from '@vllnt/ui'
```

## Usage

```tsx
const options = [
  {
    key: "value",
    label: "Example item",
  },
]

<ViewSwitcher
  options={options}
/>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `defaultKey` | `string` | No | - |
| `options` | `ViewOption[]` | Yes | Array of `ViewOption` objects |
| `paramName` | `string` | No | - |

  label: string

| Field | Type | Required | Description |
