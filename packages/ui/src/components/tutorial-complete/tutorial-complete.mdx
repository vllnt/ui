# Tutorial Complete

Completion screen displayed when a tutorial is finished.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/tutorial-complete.json
```

## Import

```tsx
import { TutorialComplete } from '@vllnt/ui'
```

## Usage

```tsx
<TutorialComplete
  backHref="value"
  completionPercent="0"
  href="/example"
  onGoToSection={(index) => console.log(index)}
  onRestart={() => {}}
  imageSource="value"
  socialLinks={socialLinks}
  shareUrl="/example"
  title="My Title"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />
