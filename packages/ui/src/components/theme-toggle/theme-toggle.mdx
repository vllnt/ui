# Theme Toggle

Button to toggle between light and dark themes.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/theme-toggle.json
```

## Import

```tsx
import { ThemeToggle } from '@vllnt/ui'
```

## Usage

```tsx
<ThemeToggle
  dark="value"
  light="value"
  system="value"
  toggle_theme="value"
>
  ThemeToggle content
</ThemeToggle>
```
