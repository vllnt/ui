# Radio Group

Group of radio buttons for single-selection input.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/radio-group.json
```

## Import

```tsx
import { RadioGroup, RadioGroupItem } from '@vllnt/ui'
```

## Usage

```tsx
<RadioGroup>
  <RadioGroup.displayName />
</RadioGroup>
```
