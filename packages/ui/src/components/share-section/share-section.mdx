# Share Section

Social sharing buttons and link copy section.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/share-section.json
```

## Import

```tsx
import { ShareSection } from '@vllnt/ui'
```

| `shareOn` | `string` | Yes | - |
