# Terminal

Terminal-style display for command output.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/terminal.json
```

## Import

```tsx
import { Terminal, SimpleTerminal } from '@vllnt/ui'
```
