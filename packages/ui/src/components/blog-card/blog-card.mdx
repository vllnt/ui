# Blog Card

Card layout for displaying blog post previews.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/blog-card.json
```

## Import

```tsx
import { ContentCard, BlogCard } from '@vllnt/ui'
```

## Props

<Controls />

| ---- | ---- | -------- | ----------- |
| `formatDate` | `(date: string, lang: string) =&gt; string` | No | - |
| `getDictValue` | `(dict: Record&lt;string, unknown&gt;, path: string) =&gt; string` | No | - |
| `href` | `string` | Yes | - |
| `lang` | `string` | No | - |
| `post` | `ContentCardPost` | Yes | - |
