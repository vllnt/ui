# Blog Card

Card layout for displaying blog post previews.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/blog-card.json
```

## Import

```tsx
import { ContentCard, BlogCard } from '@vllnt/ui'
```

## Usage

```tsx
<BlogCard
  href="/example"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `formatDate` | `(date: string, lang: string) =&gt; string` | No | Callback: `(date: string, lang: string) => string` |
| `getDictValue` | `(dict: Record&lt;string, unknown&gt;, path: string) =&gt; string` | No | Callback: `(dict: Record<string, unknown>, path: string) => string` |
| `href` | `string` | Yes | - |
| `lang` | `string` | No | - |
| `post` | `ContentCardPost` | Yes | `ContentCardPost` object |
| `readMoreLabel` | `string` | No | - |
| `showBadge` | `boolean` | No | `true` or `false` |
| `showDate` | `boolean` | No | `true` or `false` |
| `showReadMore` | `boolean` | No | `true` or `false` |
| `updatedLabel` | `string` | No | - |
