# Inline Input

Inline text input with keyboard commit and cancel support.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/inline-input.json
```

## Import

```tsx
import { InlineInput } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `className` | `string` | No | - |
| `onCancel` | `() =&gt; void` | No | - |
| `onChange` | `(value: string) =&gt; void` | Yes | - |
| `onCommit` | `(value: string) =&gt; void` | Yes | - |
| `value` | `string` | Yes | - |
