# Progress Card

Card displaying progress metrics and status.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/progress-card.json
```

## Import

```tsx
import { ContentCard } from '@vllnt/ui'
```

## Usage

```tsx
<ProgressCard
  badgeLabel="Label"
  description="A short description"
  href="/example"
  href="/example"
  metadata={metadata}
  title="My Title"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />
