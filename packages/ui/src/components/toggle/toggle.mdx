# Toggle

Two-state toggle button.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/toggle.json
```

## Import

```tsx
import { Toggle } from '@vllnt/ui'
```

## Usage

```tsx
<Toggle>
  <Toggle.displayName />
</Toggle>
```
