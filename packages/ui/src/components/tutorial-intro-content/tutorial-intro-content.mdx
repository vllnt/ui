# Tutorial Intro Content

Introduction section for tutorial pages with overview and prerequisites.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/tutorial-intro-content.json
```

## Import

```tsx
import { TutorialIntroContent } from '@vllnt/ui'
```

## Usage

```tsx
<TutorialIntroContent
  content="value"
  title="My Title"
>
  TutorialIntroContent content
</TutorialIntroContent>
```
