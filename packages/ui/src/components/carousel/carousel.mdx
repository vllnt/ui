# Carousel

Scrollable content carousel with navigation controls.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/carousel.json
```

## Import

```tsx
import { Carousel, CarouselContent, CarouselItem, CarouselNext, CarouselPrevious } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `opts` | `CarouselOptions` | No | - |
| `orientation` | `"horizontal" \| "vertical"` | No | - |
| `plugins` | `CarouselPlugin` | No | - |
| `setApi` | `(api: CarouselApi) =&gt; void` | No | - |
