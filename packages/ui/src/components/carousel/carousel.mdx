# Carousel

Scrollable content carousel with navigation controls.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/carousel.json
```

## Import

```tsx
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
} from '@vllnt/ui'
```

## Usage

```tsx
<Carousel>Carousel content</Carousel>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `opts` | `CarouselOptions` | No | - |
| `orientation` | `"horizontal" \| "vertical"` | No | `horizontal`, `vertical` |
| `plugins` | `CarouselPlugin` | No | - |
| `setApi` | `(api: CarouselApi) =&gt; void` | No | Callback: `(api: CarouselApi) => void` |

## Type Definitions

type UseCarouselLogicOptions = {
  options: CarouselOptions
  orientation: "horizontal" | "vertical"
