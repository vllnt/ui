# Table Of Contents Panel

Side panel rendering a table of contents for page navigation.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/table-of-contents-panel.json
```

## Import

```tsx
import { TableOfContentsPanel } from '@vllnt/ui'
```

## Usage

```tsx
const sections = [
  {
    id: "1",
    title: "Example item",
  },
]

<TableOfContentsPanel
  completionCount="0"
  currentSectionIndex="0"
  onClose={() => {}}
  onSelectSection={(index) => console.log(index)}
  sections={sections}
  totalSections="0"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `closeIcon` | `ReactNode` | No | React content (JSX, string, number, etc.) |
| `completedSections` | `Set&lt;string&gt;` | Yes | - |
| `completionCount` | `number` | Yes | - |
| `currentSectionIndex` | `number` | Yes | - |
| `isOpen` | `boolean` | Yes | `true` or `false` |
| `onClose` | `() =&gt; void` | Yes | Callback function |
| `onReset` | `() =&gt; void` | No | Callback function |
| `onSelectSection` | `(index: number) =&gt; void` | Yes | Callback: `(index: number) => void` |

| `resetLabel` | `string` | No | - |
| `sections` | `TOCSection[]` | Yes | Array of `TOCSection` objects |
| `title` | `string` | No | - |
| `totalSections` | `number` | Yes | - |
