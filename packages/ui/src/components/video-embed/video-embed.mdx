# Video Embed

Responsive video embed for YouTube and other providers.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/video-embed.json
```

## Import

```tsx
import { VideoEmbed } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `aspectRatio` | `"1/1" \| "4/3" \| "16/9"` | No | - |
| `src` | `string` | Yes | - |
| `thumbnail` | `string` | No | - |
| `title` | `string` | Yes | - |
| `type` | `"custom" \| "vimeo" \| "youtube"` | No | - |
