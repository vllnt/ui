# ParallelTimeline

Multi-track timeline with a shared time axis for comparing concurrent events across different tracks (civilizations, regions, domains). Renders each track as a horizontal lane with event markers positioned by year. Negative years render as `BCE`, positive as `CE`. Optional background era bands span every track.

Data-driven — pass `tracks: ParallelTimelineTrack[]` and the visible window via `startYear` / `endYear`.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/parallel-timeline.json
```

## Import

```tsx
import {
  ParallelTimeline,
  type ParallelTimelineTrack,
} from '@vllnt/ui'
```

## Usage

```tsx
<ParallelTimeline
  startYear={-500}
  endYear={500}
  tracks={[
    {
      id: 'rome',
      name: 'Rome',
      color: 'red',
      events: [
        { id: 'augustus', year: -27, title: 'Augustus becomes Emperor' },
        { id: 'fall', year: 476, title: 'Fall of Western Rome' },
      ],
    },
    {
      id: 'china',
      name: 'China',
      color: 'amber',
      events: [
        { id: 'qin', year: -221, title: 'Qin unifies China' },
        { id: 'han-end', year: 220, title: 'End of Han Dynasty' },
      ],
    },
  ]}
/>
```

## Notes

- Years use the proleptic calendar — pass negative integers for BCE, positive for CE.
- Cross-track connectors, synchronized pan/zoom, collapsible tracks, and click-to-compare are intentionally **out of scope** — drive them from consumer code via the data slots.
- Each track emits `data-track-id="<id>"` and each event emits `data-event-id="<id>"` + `data-event-year` for analytics.
