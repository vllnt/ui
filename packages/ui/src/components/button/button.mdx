# Button

Interactive button with multiple variants and sizes.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/button.json
```

## Import

```tsx
import { Button } from '@vllnt/ui'
```

## Usage

```tsx
<Button>Click me</Button>
```
