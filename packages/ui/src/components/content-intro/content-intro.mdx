# Content Intro

Introductory section for content pages with title and description.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/content-intro.json
```

## Import

```tsx
import { ContentIntro } from '@vllnt/ui'
```

## Usage

```tsx
const sections = [
  {
    id: "1",
    title: "Example item",
  },
]

<ContentIntro
  estimatedTime="value"
  onGoToSection={(index) => console.log(index)}
  onStart={() => {}}
  renderIntroContent={() => {}}
  sections={sections}
  title="My Title"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `additionalContent` | `ReactNode` | No | React content (JSX, string, number, etc.) |
| `completedSections` | `Set&lt;string&gt;` | Yes | - |
| `estimatedTime` | `string` | Yes | - |
| `isLoading` | `boolean` | No | `true` or `false` |
| `labels` | `ContentIntroLabels` | No | `ContentIntroLabels` object |
| `onGoToSection` | `(index: number) =&gt; void` | Yes | Callback: `(index: number) => void` |
| `onStart` | `() =&gt; void` | Yes | Callback function |
| `renderIntroContent` | `() =&gt; ReactNode` | Yes | Callback function |
| `sections` | `ContentIntroSection[]` | Yes | Array of `ContentIntroSection` objects |
| `title` | `string` | Yes | - |

## Type Definitions
