# Popover

Floating content panel anchored to a trigger element.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/popover.json
```

## Import

```tsx
import {
  Popover,
  PopoverAnchor,
  PopoverContent,
  PopoverTrigger,
} from '@vllnt/ui'
```

## Usage

```tsx
<Popover>Popover content</Popover>
```
