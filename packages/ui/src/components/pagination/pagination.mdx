# Pagination

Page navigation controls with previous, next, and page links.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/pagination.json
```

## Import

```tsx
import { Pagination } from '@vllnt/ui'
```

## Usage

```tsx
<Pagination
  baseUrl="/example"
  currentPage="0"
  totalPages="0"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />
