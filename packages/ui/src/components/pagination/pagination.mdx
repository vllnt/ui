# Pagination

Page navigation controls with previous, next, and page links.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/pagination.json
```

## Import

```tsx
import { Pagination } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `baseUrl` | `string` | Yes | - |
| `className` | `string` | No | - |
| `currentPage` | `number` | Yes | - |
| `totalPages` | `number` | Yes | - |
