# TLDR Section

Summary section for quick content overview.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/tldr-section.json
```

## Import

```tsx
import { TLDRSection } from '@vllnt/ui'
```

## Usage

```tsx
<TldrSection
  label="Label"
>
  TldrSection content
</TldrSection>
```

<Canvas of={Stories.Default} sourceState="shown" />
