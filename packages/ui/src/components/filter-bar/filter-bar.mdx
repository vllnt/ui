# Filter Bar

Horizontal bar with filter controls for content lists.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/filter-bar.json
```

## Import

```tsx
import { FilterBar } from '@vllnt/ui'
```

## Usage

```tsx
<FilterBar
  currentDifficulty="value"
  currentTags={currentTags}
  onFiltersChange={() => {}}
  searchQuery="value"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />
