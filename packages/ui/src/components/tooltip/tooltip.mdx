# Tooltip

Informational popup displayed on hover or focus.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/tooltip.json
```

## Import

```tsx
import {
  Tooltip,
  TooltipContent,
  TooltipProvider,
  TooltipTrigger,
} from '@vllnt/ui'
```

## Usage

```tsx
<Tooltip>Hover me</Tooltip>
```
