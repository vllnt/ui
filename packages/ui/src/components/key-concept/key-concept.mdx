# Key Concept

Highlighted definition block for key terms and a glossary list.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/key-concept.json
```

## Import

```tsx
import { KeyConcept, Glossary } from '@vllnt/ui'
```

## Usage

```tsx
<KeyConcept
  term="value"
>
  KeyConcept content
</KeyConcept>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `highlight` | `boolean` | No | `true` or `false` |
| `icon` | `ReactNode` | No | React content (JSX, string, number, etc.) |
| `term` | `string` | Yes | - |
