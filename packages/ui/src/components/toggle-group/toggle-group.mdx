# Toggle Group

Group of toggle buttons for single or multiple selection.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/toggle-group.json
```

## Import

```tsx
import { ToggleGroup, ToggleGroupItem } from '@vllnt/ui'
```

## Usage

```tsx
<ToggleGroup>
  <ToggleGroup.displayName />
</ToggleGroup>
```
