# Command

Command palette with search, groups, and keyboard navigation.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/command.json
```

## Import

```tsx
import {
  Command,
  CommandDialog,
  CommandEmpty,
  CommandGroup,
  CommandInput,
  CommandItem,
  CommandList,
  CommandSeparator,
  CommandShortcut,
} from '@vllnt/ui'
```

## Usage

```tsx
<Command>
  <Command.displayName />
</Command>
```
