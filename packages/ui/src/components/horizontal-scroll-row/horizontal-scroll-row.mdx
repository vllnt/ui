# Horizontal Scroll Row

Horizontal scroll container with snap scrolling, chevron navigation, and hidden scrollbar.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/horizontal-scroll-row.json
```

## Import

```tsx
import { HorizontalScrollRow } from '@vllnt/ui'
```

## Usage

```tsx
<HorizontalScrollRow
  title="My Title"
>
  HorizontalScrollRow content
</HorizontalScrollRow>
```
