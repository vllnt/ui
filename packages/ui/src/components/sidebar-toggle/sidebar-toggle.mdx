# Sidebar Toggle

Responsive toggle button for opening and closing the sidebar.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/sidebar-toggle.json
```

## Import

```tsx
import { SidebarToggle } from '@vllnt/ui'
```

## Usage

```tsx
<SidebarToggle
  onToggle={() => {}}
  open
/>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `onToggle` | `() =&gt; void` | Yes | Callback function |
| `open` | `boolean` | Yes | `true` or `false` |
