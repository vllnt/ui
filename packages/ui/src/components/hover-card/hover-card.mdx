# Hover Card

Card that appears on hover for previewing content.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/hover-card.json
```

## Import

```tsx
import { HoverCard, HoverCardContent, HoverCardTrigger } from '@vllnt/ui'
```

## Usage

```tsx
<HoverCard>...</HoverCard>
```
