# Tutorial MDX

MDX renderer tailored for tutorial content with custom components.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/tutorial-mdx.json
```

## Import

```tsx
import { TutorialMDX } from '@vllnt/ui'
```

## Usage

```tsx
<TutorialMdx
  content="value"
>
  TutorialMdx content
</TutorialMdx>
```
