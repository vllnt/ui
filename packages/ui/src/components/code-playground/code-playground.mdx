# Code Playground

Interactive code editor with live preview.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/code-playground.json
```

## Import

```tsx
import { CodePlayground, FileTree } from '@vllnt/ui'
```

| `description` | `string` | No | - |
| `filename` | `string` | No | - |
| `highlightLines` | `number[]` | No | - |
| `language` | `string` | No | - |
| `showLineNumbers` | `boolean` | No | - |
| `title` | `string` | Yes | - |
