# Spinner

Animated loading spinner indicator.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/spinner.json
```

## Import

```tsx
import { Spinner } from '@vllnt/ui'
```

## Usage

```tsx
<Spinner>Spinner content</Spinner>
```
