# Table

Styled data table with header, body, and footer sections.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/table.json
```

## Import

```tsx
import {
  Table,
  TableBody,
  TableCaption,
  TableCell,
  TableFooter,
  TableHead,
  TableHeader,
  TableRow,
} from '@vllnt/ui'
```

## Usage

```tsx
<Table>Table content</Table>
```
