# Sheet

Slide-over panel from the edge of the screen.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/sheet.json
```

## Import

```tsx
import {
  Sheet,
  SheetClose,
  SheetContent,
  SheetDescription,
  SheetFooter,
  SheetHeader,
  SheetOverlay,
  SheetPortal,
  SheetTitle,
  SheetTrigger,
} from '@vllnt/ui'
```

## Usage

```tsx
<Sheet>Sheet content</Sheet>
```
