# Alert

Displays an alert message to the user.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/alert.json
```

## Import

```tsx
import { Alert, AlertDescription, AlertTitle } from '@vllnt/ui'
```

## Usage

```tsx
<Alert>This is an alert message.</Alert>
```
