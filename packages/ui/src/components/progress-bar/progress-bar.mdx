# Progress Bar

Visual progress indicator with labels and completion state.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/progress-bar.json
```

## Import

```tsx
import { ProgressBar } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `className` | `string` | No | - |
| `completedLabel` | `string` | No | - |
| `currentLabel` | `string` | No | - |
| `isComplete` | `boolean` | No | - |
| `isLoading` | `boolean` | No | - |
| `max` | `number` | Yes | - |
| `showLabels` | `boolean` | No | - |
| `value` | `number` | Yes | - |
