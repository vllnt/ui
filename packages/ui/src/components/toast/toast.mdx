# Toast

Temporary notification messages with action support.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/toast.json
```

## Import

```tsx
import {
  Toast,
  ToastAction,
  ToastClose,
  ToastDescription,
  ToastTitle,
} from '@vllnt/ui'
```

## Usage

```tsx
<Toast>Toast content</Toast>
```
