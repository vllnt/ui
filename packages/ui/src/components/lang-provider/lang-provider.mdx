# Lang Provider

Context provider for language and internationalization.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/lang-provider.json
```

## Import

```tsx
import { LangProvider } from '@vllnt/ui'
```

## Usage

```tsx
<LangProvider
  supportedLanguages={supportedLanguages}
/>
```

<Canvas of={Stories.Default} sourceState="shown" />
