# Scroll Area

Custom scrollable area with styled scrollbars.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/scroll-area.json
```

## Import

```tsx
import { ScrollArea, ScrollBar } from '@vllnt/ui'
```

## Usage

```tsx
<ScrollArea>
  <ScrollArea.displayName />
</ScrollArea>
```
