# Profile Section

User profile display section with avatar and details.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/profile-section.json
```

## Import

```tsx
import { ProfileSection } from '@vllnt/ui'
```

## Usage

```tsx
const socialLinks = [
  {
    href: "/example",
    label: "Example item",
  },
]

<ProfileSection
  socialLinks={socialLinks}
/>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `compact` | `boolean` | No | `true` or `false` |
| `dict` | `ProfileDict` | Yes | `ProfileDict` object |
| `imageAlt` | `string` | No | - |
| `imageSource` | `string` | No | - |
| `socialLinks` | `SocialLink[]` | No | Array of `SocialLink` objects |

## Type Definitions

### ProfileDict

```tsx
type ProfileDict = {
  profile: {
  name: string
  tagline: string
}
```

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `profile` | `{` | Yes | - |
| `name` | `string` | Yes | - |
| `tagline` | `string` | Yes | - |

### SocialLink

```tsx
type SocialLink = {
  href: string
  label: string
}
```

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `href` | `string` | Yes | - |
| `label` | `string` | Yes | - |
