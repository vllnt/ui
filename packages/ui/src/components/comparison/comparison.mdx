# Comparison

Side-by-side comparison layout for content or features.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/comparison.json
```

## Import

```tsx
import { Comparison, BeforeAfter } from '@vllnt/ui'
```

## Usage

```tsx
<Comparison />
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `after` | `ComparisonSide` | Yes | `ComparisonSide` object |
| `before` | `ComparisonSide` | Yes | `ComparisonSide` object |
| `title` | `string` | No | - |

## Type Definitions

### ComparisonSide

```tsx
type ComparisonSide = {
  items: string[]
  title: string
  variant?: "bad" | "good" | "neutral"
}
```

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `items` | `string[]` | Yes | Array of `string` |
| `title` | `string` | Yes | - |
| `variant` | `"bad" \| "good" \| "neutral"` | No | `bad`, `good`, `neutral` |

### BeforeAfterProps

```tsx
type BeforeAfterProps = {
  after: ReactNode
  before: ReactNode
  title?: string
}
```

| Field | Type | Required | Description |
| ----- | ---- | -------- | ----------- |
| `after` | `ReactNode` | Yes | React content (JSX, string, number, etc.) |
| `before` | `ReactNode` | Yes | React content (JSX, string, number, etc.) |
| `title` | `string` | No | - |
