# Accordion

Collapsible content sections supporting single or multiple open items.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/accordion.json
```

## Import

```tsx
import { Accordion, AccordionContent, AccordionItem, AccordionTrigger } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `children` | `ReactNode` | Yes | - |
| `className` | `string` | No | - |
| `defaultValue` | `string \| string[]` | No | - |
| `type` | `"multiple" \| "single"` | No | - |
