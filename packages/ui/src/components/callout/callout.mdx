# Callout

Highlighted content block with variant styles for info, warning, danger, and more.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/callout.json
```

## Import

```tsx
import { Callout } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `children` | `ReactNode` | Yes | - |
| `className` | `string` | No | - |
| `icon` | `ReactNode` | No | - |
| `title` | `string` | No | - |
| `variant` | `CalloutVariant` | No | - |
