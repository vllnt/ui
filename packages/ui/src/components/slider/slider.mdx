# Slider

Range slider input for selecting numeric values.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/slider.json
```

## Import

```tsx
import { Slider } from '@vllnt/ui'
```

## Usage

```tsx
<Slider>
  <Slider.displayName />
</Slider>
```
