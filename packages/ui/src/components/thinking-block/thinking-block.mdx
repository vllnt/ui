# Thinking Block

Collapsible block showing AI thinking/reasoning with streaming support.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/thinking-block.json
```

## Import

```tsx
import { ThinkingBlock } from '@vllnt/ui'
```

## Usage

```tsx
<ThinkingBlock
  thinking="value"
/>
```

<Canvas of={Stories.Default} sourceState="shown" />

## API Reference

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `isStreaming` | `boolean` | No | `true` or `false` |
| `thinking` | `string` | Yes | - |
