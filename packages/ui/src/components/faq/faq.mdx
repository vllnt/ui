# FAQ

Frequently asked questions section with expandable answers.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/faq.json
```

## Import

```tsx
import { FAQ, FAQItem } from '@vllnt/ui'
```

## Usage

```tsx
<Faq
  question="What is React?"
>
  Faq content
</Faq>
```

<Canvas of={Stories.Default} sourceState="shown" />
