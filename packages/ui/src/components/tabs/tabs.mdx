# Tabs

Tabbed content panels with keyboard-accessible tab triggers.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.com/r/tabs.json
```

## Import

```tsx
import { Tabs, TabsContent, TabsList, TabsTrigger } from '@vllnt/ui'
```

## Props

<Controls />

| Prop | Type | Required | Description |
| ---- | ---- | -------- | ----------- |
| `children` | `ReactNode` | Yes | - |
| `className` | `string` | No | - |
| `defaultValue` | `string` | Yes | - |
| `onValueChange` | `(value: string) =&gt; void` | No | - |
