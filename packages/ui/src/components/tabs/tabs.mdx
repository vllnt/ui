# Tabs

Tabbed content panels with keyboard-accessible tab triggers.

<Primary />

## Installation

```bash
pnpm dlx shadcn@latest add https://ui.vllnt.ai/r/tabs.json
```

## Import

```tsx
import {
  Tabs,
  TabsContent,
  TabsList,
  TabsTrigger,
} from '@vllnt/ui'
```

## Usage

```tsx
<Tabs defaultValue="value">
  <Tabs.List />
  <Tabs.Trigger value="item-1" />
  <Tabs.Content value="item-1" />
</Tabs>
```
