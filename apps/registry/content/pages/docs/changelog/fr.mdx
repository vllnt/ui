# Changelog

Cette page inclut les changelogs du depot et du package au moment de la compilation. Le changelog du package publie reste la source des notes de version au niveau du package, tandis que le changelog racine suit les changements du site, du registre, de la CI et des surfaces destinees aux agents.

## Sources

- `CHANGELOG.md` racine : changements a l'echelle du depot et cible de version actuelle.
- `packages/ui/CHANGELOG.md` : changements publies du package `@vllnt/ui`.
- GitHub Releases : artefacts de version et notes d'annonce.
