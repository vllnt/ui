Applies the persisted theme preset on mount and injects a no-flash script so the saved preset is set before paint. Part of the utility family in VLLNT UI, it ships as a
machine-readable registry entry — copy the source directly into your app with
the shadcn CLI and own it, no runtime dependency on a component library.
