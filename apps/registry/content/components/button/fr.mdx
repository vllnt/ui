Le Button est le controle interactif principal de VLLNT UI — un composant React
accessible et navigable au clavier, avec des variantes (primaire, secondaire,
outline, ghost, destructive, lien) et des tailles. Il est publie comme entree de
registre lisible par machine : vous installez la source directement dans votre
application et vous la possedez.
