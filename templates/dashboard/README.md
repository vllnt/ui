# Dashboard

Dense dashboard kit with sidebar navigation, filters, data tables, chart panels, and status-heavy operational views.

## Install

```bash
pnpm dlx create-vllnt-app@latest --template dashboard
```

## Included pages

- Overview
- Reports
- Users
- Operations
- Settings
