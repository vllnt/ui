# Docs Site

Documentation site kit for libraries that want VLLNT-style docs, navigation, search, code samples, and component pages.

## Install

```bash
pnpm dlx create-vllnt-app@latest --template docs-site
```

## Included pages

- Docs home
- Guide
- Components
- Reference
- Changelog
