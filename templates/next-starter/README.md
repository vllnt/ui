# Next Starter

App Router starter with auth-ready routes, theming, and five example pages wired around VLLNT UI primitives.

## Install

```bash
pnpm dlx create-vllnt-app@latest --template next-starter
```

## Included pages

- Home
- Dashboard
- Account
- Settings
- Billing
