# SaaS

SaaS kit with landing sections, pricing, auth entry points, account billing, and a dashboard shell.

## Install

```bash
pnpm dlx create-vllnt-app@latest --template saas
```

## Included pages

- Landing
- Pricing
- Sign in
- Onboarding
- Dashboard
